# Security Policy

## Supported Versions

Sử dụng phần này để thông báo cho mọi người biết phiên bản nào của dự án hiện đang được hỗ trợ các bản cập nhật bảo mật.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

Sử dụng phần này để thông báo cho mọi người biết cách báo cáo lỗ hổng bảo mật.

Nếu bạn phát hiện lỗ hổng bảo mật, vui lòng báo cáo bằng cách tạo Issue trên Github hoặc gửi email trực tiếp cho chúng tôi (xem trong `README.md`). Chúng tôi sẽ cố gắng phản hồi và khắc phục sớm nhất có thể.
